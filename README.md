# BODOT
A simple little build tool for godot projects

## Commands
To view current config info
`bodot --info`

To build specific export
`bodot --build windows|html5|linux|android`

To build all available exports
`bodot --build --all`

To promote patch
`bodot --patch --up 1`

To demote patch
`bodot --patch --down 1`

To promote minor
`bodot --minor --up 1`

To demote minor
`bodot --minor --down 1`

To promote major
`bodot --major --up 1`

To demote major
`bodot --major --down 1`

To set the meta
`bodot --meta alpha|beta|etc`
